### 9. 数值统计

**题干**

输入若干组数据，每组先输入一个整数 $n$，随后输入 $n$ 个实数。统计这 $n$ 个数中**负数、零、正数**的个数，依次输出。

当输入的 $n = 0$ 时，程序结束。

**输入**

多组输入。每组两行：
- 第一行一个整数 $n$；
- 第二行 $n$ 个浮点数（空格分隔）。

当 $n = 0$ 时结束输入。

**输出**

每组输出一行，三个整数（空格分隔）：负数个数、零的个数、正数个数。

**样例 1**

```
输入：
6
0 1 2 3 -1 0
0

输出：
1 2 3
```

**样例 2**

```
输入：
5
-3.5 -2.1 0 0 4.8
7
-1 -1 -1 -1 -1 -1 -1
0

输出：
2 1 2
7 0 0
```
