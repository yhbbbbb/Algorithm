# 个人算法学习仓库
## 🏠 仓库介绍
欢迎来到我的算法学习仓库！

本仓库主要用于**归档日常算法练习题、课程实验代码、经典算法实现以及学习笔记**。
记录学习过程中掌握的各类算法思想、编码实践与解题思路，核心目的是整理知识、留存代码，方便日后复习、复盘与查漏补缺。

## 📚 内容分类
按照学习方向与课程模块划分，目录结构如下：
1. **基础算法**
   包含基础排序、查找、递归、枚举等入门算法及课后练习题。
2. **图算法**
   图的遍历（BFS/DFS）、最短路径（Dijkstra、Floyd、A*）等图论算法与实验。
3. **搜索算法**
   回溯法、分支限界法等组合优化算法，包含装载问题、0-1背包、最大团等经典例题。
4.  **实验文档**
   课程实验报告、算法原理总结、解题思路分析等配套文档。

## 💻 运行环境
- 编程语言：Python 3.7+、C++

## ✍️ 代码规范
1. 所有代码添加中文注释，标注算法思路、核心逻辑；
2. 每道习题/实验均内置测试用例，可直接运行验证结果；
3. 目录、文件统一命名，分类清晰，方便检索。
