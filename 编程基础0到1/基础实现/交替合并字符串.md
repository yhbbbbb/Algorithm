### 交替合并字符串

**难度：** 🟢 简单

**题目描述**

给你两个字符串 `word1` 和 `word2`。请你从 `word1` 开始，通过交替添加字母来合并字符串。如果一个字符串比另一个字符串长，就将多出来的字母追加到合并后字符串的末尾。

返回**合并后的字符串**。

**输入格式**

- 第一行：字符串 `word1`
- 第二行：字符串 `word2`

**输出格式**

输出一行，为交替合并后的字符串。

**样例 1**

```
输入：
abc
pqr

输出：
apbqcr
```

> 解释：字符串合并情况如下所示：
> word1：  a   b   c
> word2：    p   q   r
> 合并后：  a p b q c r

**样例 2**

```
输入：
ab
pqrs

输出：
apbqrs
```

> 解释：word2 比 word1 长，"rs" 追加到末尾。

**提示**

- $1 \le$ `word1.length`, `word2.length` $\le 100$
- `word1` 和 `word2` 由小写英文字母组成
